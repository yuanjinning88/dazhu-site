# 重大方向调整（同日 19:00 起）

## 决策背景

用户反馈后，将整个项目从**暗色作品集网站**转为**Apple 白底风个人内容站**。

## 架构变更

| 变更项 | 旧 | 新 |
|--------|----|----|
| 配色 | 黑底 #0D0D0D + 橙+青蓝 | 白底 #FFFFFF + Apple 蓝 #0071E3 |
| 语言 | 中英双语 i18n | 纯中文，移除 react-i18next |
| 内容 | 9 个履历 Section | 随笔/音乐/电影/笔记/照片 5 大板块 |
| 数据 | 本地 TS/MD 文件 | Supabase 云端数据库 |
| 导航 | 锚点滚动 | React Router 多页面 |
| 图标 | 32 个 SVG | 保留 Icon 组件 |
