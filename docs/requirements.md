# 产品需求文档 v2

## 项目概述

**大猪**的个人内容站 — 记录音乐、电影、笔记、随笔与照片。

## 核心定位

- Apple 白底极简风格，克制的交互，不抢内容的注意力
- 云端数据库驱动，所有人可访问
- 网页上直接添加内容，无需写代码

## 内容板块

| 板块 | 说明 | 数据来源 | 添加方式 |
|------|------|----------|----------|
| 首页 | 各板块最新内容聚合 | Supabase | — |
| 随笔 | 文章列表 + 详情，分类筛选 | Markdown 文件 | 新增 .md 文件 |
| 音乐 | 专辑卡片网格 + 详情 + 备注 | Supabase | 网页弹窗表单 |
| 电影 | 电影卡片网格 + 详情 + 备注 | Supabase | 网页弹窗表单 |
| 笔记 | 条目列表 + 详情 | Supabase | 网页弹窗表单 |
| 照片 | 照片网格 | Supabase | 网页弹窗表单 + 粘贴图片 URL |
| 关于 | 个人简介 | 静态页面 | 编辑代码 |

## 封面策略

- 音乐：iTunes API → MusicBrainz 双重搜索
- 电影：TMDB API 搜索
- 匹配不到 → 确定性渐变色兜底（同名称同颜色）

## 导航

- 顶部导航栏：首页/随笔/音乐/电影/笔记/照片
- 左侧汉堡菜单：关于（可扩展书单、友链）
- 移动端：顶部汉堡菜单
