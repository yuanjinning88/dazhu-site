## 为什么做个人网站？

在社交媒体泛滥的时代，拥有一个**属于自己的角落**变得愈发重要。不受算法控制，不受平台限制。

这就是我搭建这个网站的初衷。

## 设计原则

我给自己定了几条规则：

- **克制**：不堆砌效果，每个像素都有存在的理由
- **顺滑**：动画是为了引导注意力，不是为了炫技
- **诚实**：不展示虚假的数字，不夸大经历

## 技术选型

| 需求 | 选择 | 原因 |
|------|------|------|
| 框架 | React | 生态成熟，Framer Motion 做动画一流 |
| 构建 | Vite | 快到没有感知的开发体验 |
| 样式 | Tailwind CSS | 约束即自由 |
| 动画 | Framer Motion | 苹果级别交互的基石 |
| 内容 | Markdown | 纯文本，永远可控 |

## 最难的决策

不是技术选型，而是**文案**。

用中文还是英文？Slogan 写什么？"关于我"怎么写才不显得装？

最终选择了中英双语。Slogan 定为："用代码雕刻时间，让思想自有回响。"

## 写在最后

这个网站会持续迭代。它不只是一个展示窗口，更是我思考的延伸。

如果你也在考虑搭建个人网站，我的建议是：**先上线，再完美。**
